3.2.- Clustering

Este apartado describe cómo utilizar el módulo de Clustering de EPA Explorer.

Entrenamiento

Para realizar un entrenamiento de un sistema de clustering en EPA Explorer:

  1. El usuario selecciona la opción de Entrenamiento en la pestaña Clustering del menú principal.
  2. En la vista que se presenta, el usuario podrá configurar varios parámetros de entrada para el algoritmo, como son:
    • El ejercicio en estudio.
    • El conjunto de atributos a incorporar en el estudio.
    • Parámetros propios del algoritmo de agrupación como el número de conjuntos, el número de intervalos para discretizar variables numéricas, o el número máximo de iteraciones del algoritmo.
  3. Configurados todos los parámetros, pulsar sobre el botón "Ejecutar" para lanzar la ejecución del algoritmo.
  4. Una vez el algoritmo termine su ejecución, almacenara el resultado para su posterior visualización.

Visualización

Para visualizar el resultado de la ejecución previa de un sistema de clustering en EPA Explorer:

  1. El usuario selecciona la opción de Visualizar en la pestaña Clustering del menú principal.
  2. En la vista que se presenta, el usuario podrá seleccionar el fichero almacenado con una ejecución anterior del algoritmo de Clustering.
  3. El sistema carga la información del sistema de agrupación y muestra una representación visual de los mismos.
  4. El usuario puede alternar a una vista tabular de los datos, donde podría hacer una extracción en Excel o PDF de los mismos.
