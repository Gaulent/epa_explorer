2.2.- Instalación

Este apartado describe el procedimiento de instalación de una instancia de EPA Explorer, así como los requisitos software.

Notas: Bajo la carpeta installers del CD incluido en la memoria de este Proyecto Fin de Carrera, se encuentran los ficheros de instalación tanto de R como de RStudio, necesarios para la ejecución de la herramienta.

1.-Instalación de R

El paquete de instalación de Windows se encuentra disponible en el siguiente enlace

https://cran.r-project.org/bin/windows/base/

También se encuentran disponibles versiones para los sistemas operativos Linux y Mac. 

2.- Instalación de RStudio

El paquete de instalación de Windows se encuentra disponible en el siguiente enlace

https://www.rstudio.com/products/rstudio/download/

3.- Ejecución de EPA Explorer

Puede obtener el código fuente de EPA Explorer tanto desde el CD incluido en la memoria de este Proyecto Fin de Carrera o clonando el repositorio git ubicado en la siguiente dirección.

https://bitbucket.org/gaulent/epa_explorer

A continuación, en RStudio cargaremos el proyecto desde la opción:

File -> Open Project...

En el dialogo de apertura de proyecto seleccionaremos el fichero epa_explorer.Rproj.

Una vez cargado el proyecto, podremos lanzar la ejecución del mismo con la opción "Run Shiny App" de RStudio.
