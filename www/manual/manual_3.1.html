3.1.- Análisis Exploratorio

Este apartado describe cómo utilizar el módulo de Análisis Exploratorio de Datos de EPA Explorer.

  1. Selecciona en el menú desplegable bajo la pestaña "Explorar" el tipo de visualización / análisis deseado.
  2. En la vista que se presenta seleccionar los datos de entrada para la visualización, generalmente el ejercicio de estudio y el conjunto de atributos a analizar.
  3. El sistema carga la información relevante a dicho conjunto de datos y los presenta al usuario.
  4. En ese momento el usuario puede interactuar con la representación seleccionando varios parámetros, como son:
    • Cambios en la escala.
    • Grano de la representación.
    • Jitter.
  5. El sistema reacciona en vivo ajustando la visualización obtenida por cada selección del usuario.
