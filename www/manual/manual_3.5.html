3.5.- Actualización de datos

Este apartado describe cómo utilizar el módulo de Actualización de Datos de EPA Explorer.

  1. El usuario selecciona la pestaña Actualización del menú principal.
  2. En la vista que se presenta, la herramienta mostrará que paquetes no se han incorporado aun en la base de datos.
  3. El usuario podrá seleccionar el ejercicio a descargar e incorporar a la base de datos local.
  4. Una vez seleccionado, pulsar sobre el botón "Actualizar" para realizar la importación.
  5. Cuando aparezca la notificación de "Datos Importados" ya estarán disponibles para usar en los distintos módulos de la herramienta.
