3.4.- Generación de informes

Este apartado describe cómo utilizar el módulo de Generación de informes de EPA Explorer.

  1. El usuario selecciona la pestaña Informes del menú principal.
  2. En la vista que se presenta, el usuario podrá configurar varios parámetros de entrada para el algoritmo, como son:
    • El ejercicio en estudio.
    • La plantilla markdown del informe a generar.
  3. Configurados todos los parámetros, pulsar sobre el botón "Generar Informe" para lanzar la ejecución del algoritmo.
  4. Una vez la herramienta termine de generar el informe seleccionado, está enviará al usuario el informe generado en un formato apropiado para el propio informe.
